commit message =styleguide:
    Typ    Bedeutung
    feat    Fügt ein neues Feature hinzu
    fix    Behebt einen Bug
    docs    Änderungen an der Dokumentation
    style    Code-Formatierung (keine Logik-Änderungen)
    refactor    Code-Verbesserung ohne Änderung des Verhaltens
    perf    Performance-Optimierung
    test    Fügt Tests hinzu oder aktualisiert sie
    chore    Wartungsarbeiten, z. B. Dependency-Updates
    ci    Änderungen an CI/CD-Pipelines oder Build-Prozessen
    revert    Setzt einen vorherigen Commit zurück

    style: created folder structure
